🌍 Epic Bliss

A modern, responsive static travel and tourism website built for my mother's travel business to establish a professional online presence and redirect customers from social media and other platforms to a dedicated website.

This project was developed from scratch using HTML and CSS with a strong focus on clean design, responsiveness, and user experience. It serves as both a functional business website and a portfolio project showcasing my front-end web development skills.

✨ Features
Responsive design for desktop, tablet, and mobile devices
Modern and intuitive user interface
Full-screen hero section with immersive visuals
Smooth navigation
Organized and maintainable project structure
Fast-loading static website
Professional layout for showcasing travel services

🎯 Purpose

The primary goal of Epic Bliss is to provide a professional website for a real travel business while demonstrating modern front-end development practices. The website acts as a central platform where potential customers can learn about the business, explore travel offerings, and connect with the company.
